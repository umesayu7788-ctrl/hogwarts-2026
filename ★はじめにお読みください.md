# はじめにお読みください（新規セットアップ・かんたん）

ようこそ！このキットの始め方はとてもシンプルです。月次トークンの受け取りは不要です。

## 始め方
1. このフォルダを **Claude Code で開く**
2. チャットに **「始める」** と入力
3. 最初に **DiscordユーザーID** を聞かれます（コミュニティ会員確認のため・**トークン不要**）
   - 取得方法：PC版Discordで自分のアイコンを右クリック →「ユーザーIDをコピー」
   - （メニューに出ない場合：ユーザー設定 → 詳細設定 →「開発者モード」をON にしてから）
4. あとは画面の案内（APIキー設定・声/テーマ設定など）に沿って進めるだけ。Claudeが代わりに設定してくれます。

## GitHubで自動運用する場合（重要）
このキットは GitHub Actions で毎日自動運用されます。**自動運用を有効にするには、あなたのGitHubリポジトリに次のSecretを1つ登録**してください（「始める」の中でも案内されます）：

- **Settings → Secrets and variables → Actions → New repository secret**
  - Name：`DISCORD_USER_ID`
  - Value：あなたのDiscordユーザーID（上の3で使ったものと同じ）

→ これで GitHub 上の自動運用も「コミュニティ在籍中だけ動く」状態になります。

## 大事なこと
- コミュニティに参加している間、いつでも最新の状態で使えます（**退会すると自動で停止**します）。
- 困ったら、いつでもコミュニティ管理者にご連絡ください。
