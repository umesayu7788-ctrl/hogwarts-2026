セキュリティガイド
あなたのAPIキーと個人情報が
このシステムでどのように守られているか
「APIキーをチャットに貼り付けて大丈夫？」「Threadsアカウントの情報が外に漏れないか心配」——そんな不安をお持ちの方のために、このシステムがどのようにあなたの情報を守っているかをわかりやすく説明します。
1
APIキーはどこに保存されるの？
セットアップ中にGeminiやGitHubのAPIキーをチャットに貼り付けますが、それらは即座にあなたのPC内にある .env ファイルに書き込まれます。
あなたのPC
└── .env ← ここにだけ保存される（外には出ない）
✅ インターネット上のサーバーには一切送られません。Claude Codeはあなたのパソコン上でローカルに動いているツールなので、入力した情報がクラウドに保存されることはありません。
2
APIキーがGitHubに上がることはある？
ありません。

このシステムには .gitignore という仕組みが設定されており、.env ファイルはGitHubへのアップロード対象から自動的に除外されます。

仮に「GitHubに送る」操作をしても、APIキーが含まれる .env だけはスキップされる設計になっています。
3
AIが投稿する際、Threadsの情報は安全？
ThreadsのアクセストークN（投稿権限のカギ）も .env ファイルにのみ保存されます。AIが投稿を実行する流れはこうです：
あなたのPC内の .env からトークンを読み取る
        ↓
Threads API に直接リクエストを送る
        ↓
投稿完了
トークンがあなたのPC外に出るのは「Threads APIへのリクエスト時のみ」で、その通信はThreads公式のHTTPS（暗号化通信）で保護されています。第三者が途中で盗み見ることはできません。
4
GitHub Actionsで自動実行される場合は？
自動監視など一部の機能はGitHub Actionsで動きます。この場合、APIキーはGitHub Secretsという専用の秘密情報管理機能に保存します。
  • ログや画面上に値が表示されることは一切ない
  • リポジトリの管理者であっても、登録後に値を見返すことはできない
  • GitHubのサーバー上で暗号化されて保管される
5
他のメンバーに自分の情報が見える？
ありません。

このシステムは各メンバーが自分のGitHubリポジトリを使って運用します。他のメンバーのリポジトリにアクセスする手段はなく、投稿内容・APIキー・アカウント情報が他メンバーに見えることはありません。
6
月次認証キー（.keyファイル）の仕組み
毎月配布される認証キーファイルは：
  • .gitignore により、GitHubには絶対にアップロードされない設定
  • 有効期限が切れると自動的にシステムが停止する
  • キーファイルを他人と共有しても、GitHubリポジトリへのアクセス権がなければシステムは動かない
7
このシステムが扱うデータについて
このシステムが扱うデータの利用目的は以下のみです：
データ用途
Threads投稿内容あなたのアカウントへの投稿のみ
YouTubeの公開動画情報ネタ収集（閲覧のみ・書き込みなし）
GitHub Issuesエージェント間の作業記録（あなたのリポジトリ内のみ）
Google Sheetsのデータエンゲージメント記録（設定した場合のみ）
✅ 収集したデータを第三者に送信・販売することは一切ありません。すべてあなた自身のアカウントとリポジトリの中で完結します。
8
もしAPIキーが漏れてしまったら？
万が一の場合でも、以下の手順で即座に無効化できます：
キーの種類無効化方法
Gemini API KeyGoogle AI Studio で削除・再発行
GitHub TokenGitHub設定画面でトークンを削除
YouTube API KeyGoogle Cloud Console で削除
Threads TokenMeta Developer Portalで無効化
キーを無効化した後、新しいキーを .env に設定し直すだけで復旧できます。
