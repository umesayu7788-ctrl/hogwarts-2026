### 001: 高インプ投稿プレイブック＋ペルソナ確定ガイド

| **ファイル** | `updates/packages/001_post_playbook.md` |
| **目的** | 投稿のインプレッション・フォロー転換率を体系的に向上させる |
| **対象** | SNS発信全般（業種・ジャンル不問） |
| **前提条件** | なし |
| **配布元ファイル** | `updates/packages/001_files/_universal_post_playbook.md`<br>`updates/packages/001_files/_persona_setup_guide.md`<br>`updates/packages/001_files/affiliate_appeal_patterns.md` |

| **実証データ** | 1投稿で最高7万imp/フォロー+23<br>別投稿で1.0〜1.7万imp |
| **推奨度** | ⭐⭐⭐ 最重要・全顧客推奨 |
| **所要時間** | 30-60分（ペルソナ確定時間含む） |

#### 適用判断の基準

- ✅ **適用推奨**：
  - 投稿のインプが伸び悩んでいる
  - フォロー転換が弱い
  - ペルソナが固まってない or ぶれてる
  - 必勝テンプレを導入したい
- ⏸️ **後で良い**：
  - 直近2週間で投稿が安定して伸びている
  - 既に独自の必勝テンプレで運用中
- ❌ **適用不要**：
  - 該当なし（基本は全顧客に推奨）

## 📜 アップデート読み込み手順（Claude Code 向け）

1. このマニフェストを最初から最後まで読む
2. `## 🔢 利用可能アップデート一覧` のセクションを抽出
3. 各アップデートの「適用判断の基準」を踏まえる
4. 顧客の現状ヒアリング結果と照合
5. 「適用推奨/後で良い/適用不要」に分類して提示
6. 顧客の選択を待つ

## 📌 重複適用の防止

顧客プロジェクトに `updates/_APPLIED.md` がある場合、そこに記載された
アップデート ID（例：`001`）は既に適用済み。再適用しないこと。

`_APPLIED.md` がない場合は新規顧客と判断。
