## 🏗️ アフィリ投稿の黄金構造（ジャンル横断・必修）

参考アカウント分析から普遍的に抽出された6ステップ：

1. **共感呼びかけ・自己開示**（感情の門を開く）
2. **失敗・葛藤の告白**（信頼を作る）
3. **試行錯誤の物語**（ストーリー）
4. **五感ベースの体験描写**（仮想体験）
5. **数字・専門性で根拠補強**（権威）
6. **未来想起 + 控えめなCTA**（押し付けない誘導）

→ アフィリ投稿はこの6ステップを2投稿構造（親+ツリー1）に圧縮する。
