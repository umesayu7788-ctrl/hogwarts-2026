# 商品選定基準

> 商品リサーチ担当が参照する、**商品を選ぶための判断基準**。
> ライターが参照する「product_purchase_rules.md」とは役割を分ける：
> - 本ファイル = 商品を「選ぶ」基準（What）
> - 購買心理ルール = 読者に「響かせる」方法（How）

---

## 1. 価格帯の選定

### 低単価アフィリの鉄則

- **推奨レンジ: 500〜3,000円**
- 衝動買いの心理ハードルが最も低い価格帯
- 「ポチる」までの迷い時間が短い
- クッキー期間内（楽天30日）の購入確率が高い

### 価格帯別の特性

| 価格帯 | 特性 | 訴求の方向性 |
|---|---|---|
| 500〜1,000円 | 試し買い・プチギフト需要 | 「気軽に試せる」「失敗してもダメージ小」 |
| 1,000〜2,000円 | コスパ訴求のスイートスポット | 「この価格でこの品質」「コスパ最強」 |
| 2,000〜3,000円 | 少し背伸びの満足感 | 「ちゃんとした品質」「長く使える」 |
| 3,000円以上 | 検討時間が長くCVR低下 | **基本的に避ける**（高単価アフィリ戦略が必要） |

### 価格を軸にした商品選び

- 送料込みの総額で判断（送料別だと読者離脱）
- セール価格ではなく通常価格で評価（セール終了で広告効果消失）
- 定期的に価格変動する商品は要注意（投稿時と購入時で価格が変わる）

## 2. レビュー評価の基準

### 最低ライン

- **レビュー件数**: 10件以上（母数が少ないと評価が歪む）
- **平均評価**: ★3.5以上（楽天基準。Amazonなら4.0以上）
- **投稿期間**: 直近6ヶ月にレビューがあること（販売終了リスクを避ける）

### 積極的に選ぶべき商品

- レビュー数100件以上 + 平均★4.0以上 = 鉄板商品
- レビュー数が急増中 = トレンド商品の可能性大
- 「リピート購入」コメントが多い = 継続訴求の実績

### 避けるべき商品

- ★5とサクラレビューが混在している違和感のある商品
- 低評価レビューで「届かない」「違うものが届いた」等の物流クレームが多い
- 「ステマ疑惑」がSNSで指摘されている商品

## 3. NGカテゴリ（法令・規約リスク）

### 絶対に扱わない

| カテゴリ | 理由 |
|---|---|
| 医薬品・指定医薬部外品 | 薬機法違反リスク |
| 痩身・育毛・バストアップ | 効能謳いで行政指導リスク |
| コンタクトレンズ | 医療機器該当・景表法厳格 |
| 金融商品（投資・FX・仮想通貨） | ステマ規制+金商法 |
| アダルト・出会い系 | プラットフォーム規約違反 |
| 情報商材（高額塾・副業教材） | 景表法違反リスク |

### 慎重に扱う（条件付きOK）

| カテゴリ | 条件 |
|---|---|
| 化粧品 | 効能を謳わない。「気に入っている」程度の体験ベース |
| 健康食品・サプリ | 医薬品的効能を謳わない |
| ダイエット関連 | 「痩せる」を使わず「生活習慣の工夫」の文脈で |
| ガジェット | 過度な性能強調を避ける |

## 4. 季節性・トレンド対応

### 季節商品の先取り原則

- **実需期の2〜4週間前**に紹介開始
- 例：
  - 3月上旬 → 新生活グッズ
  - 5月中旬 → 梅雨対策・除湿
  - 7月 → 熱中症対策・冷感グッズ
  - 9月 → 乾燥対策・加湿
  - 11月 → 年末年始準備・おせち
  - 1月 → 新年の習慣化グッズ

### トレンド商品の見極め方

| シグナル | 判断 |
|---|---|
| 楽天ランキングで急上昇中 | 旬。即座に紹介価値あり |
| SNSで複数インフルエンサーが紹介 | すでに飽和気味。差別化が必要 |
| TVで紹介された直後 | 在庫切れリスク。購入できない投稿は無価値 |
| 3ヶ月以上ランキング上位 | 定番商品。リピート訴求が効く |

### 避けるべきトレンド

- 一発屋のバズ商品（1週間で熱が冷めるもの）
- TikTokで爆発的に拡散しすぎた商品（偽物・類似品が大量発生）

## 5. 商品タイプ別の訴求しやすさ

### 訴求しやすい商品（推奨）

- **日用消耗品**: 繰り返し購入が発生・在庫切れリスク低
- **キッチン・収納グッズ**: Before/After訴求が効く
- **季節家電**: 時期が明確で危機感訴求が可能
- **食品・ドリンク**: 体験談ベースで語りやすい
- **文房具・PC周辺機器**: レビュー数が集まりやすくCVRが高い

### 訴求しにくい商品（避ける）

- ファッション（サイズ・好みで返品率高）
- 高額家電（検討期間長くクッキー期間切れ）
- ブランド品（並行輸入・偽物リスク）
- 定期縛り商品（読者の反発を招く）

---

## 6. 楽天アフィリエイト特有の選定ポイント

### ランキングAPIの賢い使い方

- 総合ランキングより**ジャンル別ランキング**の方が埋もれた良品を見つけやすい
- 「新着」と「急上昇」を併用してトレンドを拾う
- レビュー数でソートすると信頼性の高い商品が見つかる

### 楽天で特に売れやすいカテゴリ

- 食品（特にお米・ふるさと納税系）
- 日用品（洗剤・ティッシュ等の消耗品）
- キッチン雑貨
- 収納用品

### 楽天で避けた方が良いカテゴリ

- 高額家電（価格比較で他に流れる）
- 最新ガジェット（Amazonの方が強い）

---

## 7. Amazonアフィリエイト特有の選定ポイント（将来有効化時）

### Amazonで強いカテゴリ

- 書籍・Kindle本
- 家電・PC周辺機器
- ゲーム・玩具
- Amazon限定商品

### Amazonで注意するポイント

- **クッキー期間が24時間のみ**（楽天の30日より短い）
- そのため「今すぐ欲しい」需要の商品が向く
- 比較検討期間が長い商品（家電等）は不向き

## 8. 商品選定のチェックリスト（商品リサーチ担当が毎回確認）

商品を選ぶ前に以下を全てYESで答えられるか確認：

- [ ] 価格は500〜3,000円の範囲内か
- [ ] レビュー10件以上、★3.5以上か
- [ ] NGカテゴリに該当しないか
- [ ] 季節・時期と整合しているか
- [ ] ランキング上位 or 急上昇中 or 定番か
- [ ] 在庫が十分にあるか（在庫切れで広告が死なないか）
- [ ] 類似品・偽物が横行していないか
- [ ] 自分のターゲット読者に合っているか

**1つでもNOがあれば選定しない。**

---

## 9. 商品の優先順位付け

同条件の複数商品から選ぶ際の優先度：

1. **自分が実際に使ったことがある** > 最優先
2. **信頼できる知人が推奨している** > 次点
3. **レビュー数が圧倒的に多い** > 鉄板評価
4. **新着で勢いがある** > 先行者利益
5. **単に検索で上位に出た** > 最下位（体験がないため訴求が弱い）
