# 6つの教育フレームワーク

> フォロワーを「認知 → 興味 → 信頼 → 購入 → 継続」へ導くための6要素。
> 一方的な商品紹介ではフォロワーは動かない。読者の「価値観そのもの」を
> 育ててから提案することで、CVR（購入率）とLTV（顧客生涯価値）が上がる。

**ファネル設計担当**: 1週間で6要素全てを最低1回ずつカバーする。
**ライター**: 各投稿がどの教育要素に該当するかを意識して書く。

---

## 教育①: 目的の教育

### 目的
読者の「何のために？」を明確にする。表面的な欲求の奥にある本質的なゴールを言語化する。

### 投稿で扱うべき内容
- 「この商品を使うことで、本当に得たい未来は何か」を問いかける
- 単なる機能紹介ではなく「その先の生活の変化」を描く
- 読者自身が気づいていなかった理想像を提示する

### 投稿例のパターン
- 「時短グッズを買う本当の目的は、時間じゃなくて余白だよね」
- 「ダイエットの目的は痩せることじゃなくて、鏡を見て笑顔になる自分だよね」
- 「高機能な道具を揃える理由は、自分を大切にしてる証明だよね」

### 使う感情フック
**共感 + 深い気づき**

---

## 教育②: 信念・価値観の教育

### 目的
読者と自分の価値観を共有する。「この発信者は自分と同じ世界を見ている」と感じてもらう。

### 投稿で扱うべき内容
- 「自分はこう考える」という価値観を明確に宣言する
- 世の中の常識に対して自分なりの見解を示す
- 「こういう人を応援したい」というターゲット像を語る

### 投稿例のパターン
- 「安いから買う、は結局高くつく。私は長く使えるものを選ぶ」
- 「〇〇が流行ってるけど、私は△△の方が本質だと思う」
- 「忙しい人こそ、ちゃんとしたものを使うべきなんだよね」

### 使う感情フック
**共感 + 価値観の一致**

---

## 教育③: 問題の教育（現状認識）

### 目的
読者に「今のままではマズい」と気づかせる。問題を言語化して自覚させる。

### 投稿で扱うべき内容
- 読者が無自覚に抱えている問題を鏡のように映し出す
- 「その状態が続くとどうなるか」の未来を具体的に描く
- 問題から目を背けていることの機会損失を伝える

### 投稿例のパターン
- 「毎朝のバタバタ、実は〇〇が原因だったりする」
- 「『今はまだ大丈夫』って言ってる間に、差はどんどん開いてる」
- 「便利なはずのスマホで、むしろ時間を失ってない？」

### 使う感情フック
**危機感 + 気づき**

---

## 教育④: 手段・解決策の教育

### 目的
問題の解決方法を「選択肢」として提示する。具体的なハウツー・ノウハウ・商品を紹介する文脈を作る。

### 投稿で扱うべき内容
- 問題を解決するための複数の方法を比較する
- それぞれの方法のメリット・デメリットを公平に伝える
- 「自分はこの方法を選んでいる」という立場を示す

### 投稿例のパターン
- 「この悩みを解決する方法は3つ。一番コスパ良いのは…」
- 「人によって合う合わないがある。自分は〇〇派」
- 「完璧を目指さなくていい。まずはこれだけやればOK」

### 使う感情フック
**好奇心 + 実用性**

---

## 教育⑤: 投資・行動の教育

### 目的
「買う・変える・始める」ことへの心理的ハードルを下げる。「今の自分に必要な投資」という認識を育てる。

### 投稿で扱うべき内容
- 支出ではなく「未来への投資」というフレーミングを使う
- 金額の大小ではなく「得られるリターン」を見せる
- 「後でやろう」の機会損失を伝える

### 投稿例のパターン
- 「〇〇円の投資で、毎日の〇〇時間が変わるなら安い」
- 「迷ってる時間が一番もったいない。試してダメなら次に行けばいい」
- 「未来の自分が喜ぶ選択を、今日してあげよう」

### 使う感情フック
**損失回避 + 行動喚起**

---

## 教育⑥: 継続・習慣の教育

### 目的
一度の購入で終わらせず、関係性を継続させる。リピーター・ファン化を促す。

### 投稿で扱うべき内容
- 「買った後」の使い方・活用術を共有する
- 習慣化のコツ・挫折しない工夫を提案する
- 一緒に成長していく仲間意識を醸成する

### 投稿例のパターン
- 「これを毎日続けると、3ヶ月後には〇〇が当たり前になってる」
- 「続かなくても大丈夫。やめても、また始めればいい」
- 「私もまだ道の途中。一緒に続けていこう」

### 使う感情フック
**共感 + 帰属意識**

## 週次運用の推奨配分

1週間（11投稿想定）で6つの教育要素をカバーする配分例：

| 曜日 | SLOT_1 (7時) | SLOT_2 (18時) | SLOT_3 (21時) |
|---|---|---|---|
| 月 | ②価値観 | ③問題 | - |
| 火 | ①目的 | - | ④手段 |
| 水 | ④手段 | - | **⑤投資（アフィリ可）** |
| 木 | ③問題 | - | - |
| 金 | ⑥継続 | **⑤投資（アフィリ可）** | - |
| 土 | ②価値観 | - | ①目的 |
| 日 | ⑥継続 | - | - |

### 配分の原則

- **アフィリ投稿は⑤の文脈で行う**（「投資」として位置付けてから商品提案）
- ③問題 → ④手段 → ⑤投資 の順で並べると購買まで自然に導ける
- ①目的 と ②価値観 は発信者の世界観を作る土台。週2回以上は入れる
- ⑥継続 は週末にフォロワーとの絆を深める位置で使う

## ライターへの重要指示

### 1投稿1要素の原則

- 1投稿には1つの教育要素だけを扱う
- 複数詰め込むと焦点がぼやけて刺さらない

### 商品紹介は⑤の後でのみ

- ①〜④だけで商品紹介をするのは禁止（売り込み感が出る）
- ⑤の文脈（投資として必要）を作ってから商品を提案する

### 読者を置き去りにしない

- 難しい言葉・専門用語を避ける
- 「知ってて当然」の態度を取らない
- 一人の友人に話すように書く

## 分析官への指標

各教育要素の投稿がどの指標で成果を出しているか記録：

| 教育要素 | 成功指標 |
|---|---|
| ①目的 | いいね数・保存数 |
| ②価値観 | フォロワー増加数 |
| ③問題 | 返信・リプライ数 |
| ④手段 | リポスト数・保存数 |
| ⑤投資 | アフィリクリック数・購入数 |
| ⑥継続 | エンゲージメント率・リピート反応 |

要素別の成果を見ることで、**どの教育が弱いか**を特定して補強する。
