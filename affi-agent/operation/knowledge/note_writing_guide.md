# 有料note作成ナレッジ — 無料導入部の教育設計 & 構成テクニック

---

## A. ビジネスにおける6つの教育（無料部分で実装する）

セールスではなく「教育＝伝える」ことで、読者が自然に購入へ至る仕組み。
無料導入部でこの6つを段階的に織り込む。

| # | 教育 | 目的 | 無料部分での実装方法 |
|---|------|------|----------------------|
| 1 | **信用の教育** | 「この人は信頼できる」と思わせる | 実績データ提示（16,503閲覧/いいね60）、失敗談の共有、専門知識の平易な説明 |
| 2 | **目的の教育** | 読者自身の目的を明確化させる | 「あなたも○○したくないですか？」→ 商品が手段として機能すると認識させる |
| 3 | **問題の教育** | 現状放置の危機感を喚起する | 「このまま月200ドル払い続けますか？」→ 放置時の悪影響を言語化 |
| 4 | **手段の教育** | この記事がベストな解決策と証明 | 他の方法との差異、成功理由、具体的な実績事例の提示 |
| 5 | **投資の教育** | 価格を"コスト"→"投資"に再定義 | 独学にかかる時間/費用と比較、決断による未来の変化を描写 |
| 6 | **行動の教育** | 今すぐ購入を促す | 限定性（段階値上げ）、購入後に得られるものの明確化 |

**核心**: 6つの教育は"売る前の準備"。売り込まず「伝える」ことに徹すれば成約率は自然に上がる。

---

## B. マーケティングファネル設計（SATORI参考）

```
認知 → 興味 → 検討 → 購入 → 推奨
```

- **認知**: Threads毎日3投稿（自動）で接触頻度を確保
- **興味**: 投稿で共感・価値提供 → noteプロフィールリンクへ
- **検討**: note無料部分で6つの教育を実施
- **購入**: 有料ラインの先に具体的解決策
- **推奨**: 読後の満足感 → 自発的シェア

## C. 売れる有料noteの構成テンプレート

### 全体構造

| 部分 | 内容 | 役割 |
|------|------|------|
| タイトル | 数字 + ベネフィット + 好奇心 | クリック誘発 |
| 無料・冒頭 | 読者の悩みを代弁して共感 | 感情的接続（信用+問題の教育） |
| 無料・中盤 | 得られる未来像 + 著者の実績 | 信頼構築（目的+手段の教育） |
| 無料・末尾 | 有料部分の内容を箇条書き予告 | 購入の最後の一押し（投資+行動の教育） |
| --- 有料ライン --- | | |
| 有料・本編 | 具体的な体験・思考・気づき | 価値提供の核 |
| 有料・結び | ビジョン・読後の余韻 | 「面白かった」で完結 |

### 無料/有料の境界ルール

- 記事全体の**冒頭20〜30%**を無料にする
- 「結論を知りたい」と最も強く感じるタイミングで有料ラインを引く
- 無料部分だけでも「読んで良かった」と思える価値を出す（出し惜しみしない）
- 短すぎると信頼を得られず、長すぎると有料部分の価値が薄まる

## D. 参考noteから学んだ実践テクニック

### wakabayashi0014（Threads×note販売戦略note）
- **痛みベース販売モデル**: 問題認識 → 原因分析 → 解決可能性の示唆 → 成果提示 → 有料へ
- **断定的な文体**: 「努力量じゃなく、やり方の違い」という逆説で注意を引く
- **権威性の明示**: 「フォロワー6,000人/売上130万円」等の具体数字
- **時間軸で緊迫感**: 「3ヶ月で結果は出る」→ 今行動しないと損

### ana_gon1111（Threads初心者向けnote）
- **段階的値上げモデル**: 初回10部完売 → 価格上昇 → 最終5,980円。「今が最安値」の緊迫感
- **タイトルの感情語**: 「禁断」「悪魔的裏戦略」で好奇心刺激
- **顧客の声を掲載**: 「即効性ありすぎです！」→ 社会的証明

## E. 価格設定の指針

| 段階 | 推奨価格 | 理由 |
|------|----------|------|
| 初回（実績なし） | 500〜800円 | 心理的ハードル最低。衝動買い発生帯 |
| 実績蓄積後 | 980〜1,500円 | 信頼+レビューが揃ってから |
| 段階値上げ | 初回300円→500円→980円 | 限定感・緊急性の演出 |

- 1,000円超は購入率が急落 → 初期は避ける
- 手数料: プラットフォーム10% + 決済5〜15% = 実手取り約75〜85%

## F. Threads → note導線の鉄則

1. リンクだけ貼らない → 内容を匂わせる一言を必ず添える
2. 「一部無料で読めます」で心理的ハードルを下げる
3. Threads投稿とnote記事のテーマを統一し、一貫したブランドイメージを維持
4. 記事公開後のSNS露出強化が成否を分ける（書いた後が勝負）

---

## G. 出木杉note執筆時の厳守ルール（MONETIZE_HANDOVER.mdより）

- 著者「出木杉」の一人称視点、実体験として語る
- **体験記・思考録**として完結。HOW TO（手順解説）は書かない
- 「続きはコミュニティで」等の誘導は**絶対禁止**
- テンプレートの中身・実装手順・コードは書かない
- 読後感は「面白かった。自分もやりたい」→ 読者が記事だけで100%満足できること
- 読者が自発的に著者を調べる → コミュニティを発見、という自然な流れ
