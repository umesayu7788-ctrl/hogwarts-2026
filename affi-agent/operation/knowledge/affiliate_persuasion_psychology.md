# 心理誘導アフィリ投稿のナレッジ（潜在意識アプローチ）

> **このナレッジはライター・校閲・コンプライアンス担当が必ず参照する。**
>
> 目標：読者が **「気づいたらリンクをタップしてた」「買ったけど後悔がない」** という状態を作る。
> 押し売り感ゼロ。商品名は最後。あくまで「自分の体験・気づきを共有する友達」として。
>
> ⚠️ **以下の例文は説明用サンプルです。**
> あなたのジャンルに合わせて言葉を置き換えて活用してください。
> 心理トリガーの「型」が普遍的なポイントなので、文章自体を真似する必要はありません。

---

## 🎯 大前提：押し売りはNG・後悔させない

### NGアプローチ（即アンフォロー誘発）
- 「絶対買って！」「これ最強！」「今すぐ！」
- 商品名を冒頭から連呼
- リンク文を煽る（「お早めに！」「在庫残りわずか！」を多用）
- ポジティブだけ書く（不自然な完璧描写）
- 「私の紹介リンクから買って」（楽天規約違反）

### OKアプローチ（自然な誘導）
- 自分の悩み・失敗から入る
- 解決のプロセスを語る
- 商品名は本文の真ん中〜後半に1度だけ
- 良い面と「気をつけたい点」を両方書く
- 「気になったら覗いてみてね」程度の誘い

---

## 🧠 8つの心理トリガー（潜在意識へのアプローチ）

### 1. ストーリーテリング（最強の誘導手段）

**人は商品ではなく「自分が変わるストーリー」を買う**

NG：
> このパジャマ、冷感素材で快適です

OK：
> 夏の夜、子どもが汗だくで何度も起きて
> 私もずっと寝不足だったんだよね
> でも、ある日素材を変えたら…
> 朝までぐっすりで、私もびっくりしちゃった

**ポイント：**
- 過去の苦悩 → 試行 → 変化のストーリー
- 「商品を買う前→買った後」の人生変化を見せる
- 数字より感覚（「ぐっすり」「肌触り」「朝の機嫌」）

---

### 2. 五感への訴求（潜在意識の入口）

**理性（脳）ではなく、感覚（体）に訴える**

抽象 → 具体に変換：
- 「気持ちいい」→「お腹のあたりがすーっとして、深呼吸しちゃう」
- 「快適」→「シーツに触れた瞬間、肩の力が抜ける」
- 「効いた」→「2日目の朝、起きた瞬間、あ、いつもと違うって思ったの」

→ 読者の脳内で**仮想体験**が起きる → 「これ私も体験したい」が無意識に発火

---

### 3. アンカリング（基準点を先に見せる）

**最初の数字が判断基準を作る**

例：
> 寝不足のママの平均睡眠時間って、4〜5時間なんだって（参照点）
> でも私、これ使い始めてから6時間以上眠れる日が増えたんだよね（変化）
> 1日90分、私の人生に時間が戻ってきた感覚（価値の翻訳）

→ 4〜5時間という数字が脳に刻まれた後、6時間という変化が大きく見える

---

### 4. 社会的証明（みんな選んでる）

**人は他人の選択を判断材料にする**

```
楽天で★4.3、レビュー2840件
「夜泣きが減った」「これで救われた」って書いてる人多くて
共働きママの間で密かに流行ってるみたい
```

**ポイント：**
- 具体的な数字（★評価・レビュー数）
- 同じ立場の人（共働きママ・夜泣きで悩んでた人）
- 「密かに」「ひっそり」の控えめワードで自然さUP

---

### 5. 損失回避（得より失うのが怖い）

**「得する」より「失わずに済む」のほうが2倍刺さる**

NG（利得フレーム）：
> これがあれば毎日6時間眠れます！

OK（損失回避フレーム）：
> 夜泣きで毎晩起きてた頃、私、子どもにイライラしちゃってたんだよね
> 今思うと、あの時の自分、可哀想だった
> もう、あの頃には戻りたくない

→ 「損失（イライラ・自己嫌悪）を避けたい」が無意識に発動

---

### 6. 失敗回避（私もこれで失敗した）

**先に失敗談を見せると、後の成功が信頼される**

```
最初、安いの買って失敗したんだよね
1ヶ月でへたってきて、結局買い直し
で、調べ直して買い直したのがこれ
最初からこっちにしておけばよかった、って思った
```

→ 「失敗してほしくない」という親心モード発動 → 自分も同じ失敗したくない → クリック

---

### 7. 未来想起（買った後の自分を見せる）

**読者の頭の中で「すでに使ってる自分」を再生させる**

> 夜中に起きずに朝までぐっすり眠れる
> 朝、子どもが「ママおはよう」って笑顔で起きてくる
> その日、自分も心に余裕があって、優しくいられる
> ……そんな日常が、月にあと10日増えるとしたら？

→ 読者の脳が**未来体験のシミュレーション**を始める → 「これは私の現実になりうる」と思った瞬間に欲しくなる

---

### 8. 専門家・権威の借用（自分の肩書き視点）

**自分の権威性を「自然に」混ぜる**（buzz_posts.mdで定義した自分の肩書き・経歴を活用）

NG（押し付けがましい）：
> {{あなたの肩書き}}{{年数}}の私が選んだ商品です！買ってください！

OK（自然に匂わせる）：
> プロでも、自分のことになると別なんだよね
> {{自分の経験領域}}でいろんな人を見てきて
> その中で「これ救世主だった」って何度も聞いた商品があって…

→ 権威を主張せず、**経験の蓄積として匂わせる**だけで信頼が乗る

## 🏗️ 黄金構造：心理誘導×参考バズ統合（7ステップ）

```
1. 共感呼びかけ・自己開示（感情の門を開く）
   「夜泣きでクタクタなママ、わかる、私もそうだったよ」

2. 失敗・葛藤の告白（信頼を作る）
   「最初、適当に買って失敗したんだよね」

3. 試行錯誤の物語（ストーリー）
   「で、調べて、見つけたのがこれ」

4. 五感ベースの体験描写（仮想体験）
   「使ってみたら、肌触りがすーっとして…」

5. 数字・専門性で根拠補強（権威）
   「楽天★4.3、{{自分の肩書き}}視点でも納得」

6. 未来想起（購入後の自分）
   「夜中に起きずに朝までぐっすりな朝、想像できる？」

7. 控えめなCTA（自由意志を残す）
   「気になったら覗いてみてね、私はここに置いとくね」
```

各ステップ、**1〜2文で簡潔に**。長文だと熱が冷める。

## 🚫 後悔買いを生まない3原則

### 原則1：商品の弱点も伝える

```
ただし、洗濯はちょっと注意が必要で
ネットに入れて優しく洗わないと、毛羽立ちやすいかも
そこさえクリアすれば、長く使えるよ
```

→ **欠点を先に出すと、長所がより信頼される**（両面提示効果）

### 原則2：価格と価値の翻訳

価格だけだと「高い」と感じる商品も、**1日あたり・月あたり**に翻訳すると見え方が変わる。

```
3,980円って一見高く感じるけど
1年使えば1日あたり10円ちょっと
コーヒー1杯より安く、夜の安心が買えるって考えるとね
```

→ ただし**誇張禁止**。実際の使用期間で計算する。

### 原則3：「自分で決めた感」を残す

```
私の話をしただけだから
合うかどうかは、ママさん自身で確かめてみてね
```

→ 押し付けられた感がない＝後悔されにくい。

## ⚠️ コンプライアンス遵守ライン

心理誘導は強力だが、以下は絶対NG：

| NG | 理由 |
|---|---|
| 「絶対治る」「100%効く」 | 景表法・薬機法違反 |
| 競合品の貶め | 名誉毀損リスク |
| 嘘の体験談（使ってないのに使った風） | ステマ規制違反 |
| 「私の紹介リンクから買って」と直接依頼 | 楽天規約違反 |
| 「在庫残り○個！急いで！」（事実でないとき） | 不当表示 |

→ 「気持ちを誘導する」のはOK、**「事実を歪める」のはNG**
