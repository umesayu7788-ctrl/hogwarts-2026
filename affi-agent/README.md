# 🎯 Affiliate Threads Agent Team

アフィリエイトマーケティング特化の Threads 自動運用AIチーム。

アフィリエージェントの6体構造をベースに、**アフィリエイト運用に必要な7体**を追加した**13体構成**。

## 🚀 クイックスタート

### 前提
- GitHub アカウント（2FA 有効化推奨）
- Google アカウント（Sheets用）
- 楽天ID（アフィリエイト登録用）
- Amazonアカウント（アソシエイト登録用）
- Threadsアカウント（アフィリ用に新規作成推奨）

### セットアップ（10分 + ナレッジ記入1時間）

1. このリポジトリを Fork または clone
2. GitHub Settings で Secrets を設定（13項目）
3. Codespaces を起動
4. ナレッジ3本を記入（最重要）
5. Claude Code で「始める」と入力

詳細は [SETUP.md](./SETUP.md) 参照。

## 📁 プロジェクト構造

```
├── .devcontainer/      # Codespaces環境定義
├── .github/
│   ├── agents/         # 13体のエージェント定義
│   └── workflows/      # GitHub Actions（cron自動実行）
├── scripts/            # Pythonスクリプト
│   └── utils/          # 共通ユーティリティ（楽天・Amazon API等）
├── operation/
│   ├── knowledge/      # ナレッジ（商品ルール・アフィリ戦略・6つの教育）
│   ├── products/       # 日次の商品リサーチ結果
│   ├── reviews/        # 口コミ分析結果
│   ├── analytics/      # 分析官レポート
│   ├── weekly/         # 週次ファネル計画
│   ├── themes/         # 今日のテーマ
│   └── memory/         # 外部メモリ
├── CLAUDE.md           # Claude Code 起動時のガイド
├── SETUP.md            # セットアップ手順
├── SECURITY.md         # セキュリティ運用ルール
└── .env.example        # 環境変数テンプレート
```

## 🔐 セキュリティ

このリポジトリは Private で運用することを**必須**とします。
APIキーは必ず GitHub Secrets で管理し、`.env` ファイルをコミットしないこと。

詳細: [SECURITY.md](./SECURITY.md)
