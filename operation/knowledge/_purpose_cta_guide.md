# 🎯 目的別CTA設計ガイド（非アフィリ・汎用版）

「焦らせない・押し付けない・自然に次の一歩を促す」という哲学は、
アフィリエイトだけでなく**あらゆるSNS発信の目的**に適用できます。

このファイルは、商品販売を目的としないアカウントのために
`affiliate_appeal_patterns.md` と等価の6パターンを再設計したものです。

---

## 🚫 絶対NG表現（目的を問わず共通）

| NG表現 | 理由 |
|---|---|
| 「今すぐDMください！」「急いで！」 | 押し付け・焦らせ感 |
| 「登録しないと損ですよ」 | 脅し感 |
| 「〇月〇日締め切り」 | 後で見返した時に古い情報 |
| 「ぜひフォローしてください」 | 乞い感・定型句感 |
| 「DMで聞かれたので置いとくね」 | 広告色強い・テンプレ感 |
| 「絶対に来てください」 | 強制感 |

## ✅ 採用するCTAパターン6種

### ① 問い合わせ・相談誘導型（サービス業向け）

**心理レバー**: 「自分から相談するのは重い → 軽く触れてみようかな」
**対象業種**: コーチング・コンサル・士業・治療家・サロン・フリーランス全般

**例文**:
```
うちではこういう相談をよく受けます。
LINEから気軽に聞いてもらえれば。

同じこと悩んでる人、ここから話しかけてみてください。（リンク）

30分だけ話してみますか？　→ プロフのリンクから。
```

**使い所**: 教育型投稿の末尾。「この悩みを持つ人 = 自分の顧客候補」という位置づけで、投稿全体が自然な営業になる。

**鉄則**: 「無料相談」よりも「気軽に」「30分だけ」「話しかけて」のほうが反応が上がる。

---

### ② コンテンツ継続型（ブランド認知・フォロー獲得）

**心理レバー**: 「次も役立ちそう → フォローしておこう」
**対象**: ブランド構築・専門家としての認知拡大・中長期のファン形成

**例文**:
```
こういうの毎週書いてます。見逃したくなければプロフから。

来週、これの応用版を書く予定。

○○について定期的に発信してます。
役に立てそうなら置いといてください。
```

**使い所**: 高インプ投稿の末尾。バズった時のフォロワー転換に使う。

**鉄則**: 「フォローしてください」と言わない。「あなたの役に立てそうなら」という文脈で提示する。

---

### ③ コミュニティ・グループ参加型

**心理レバー**: 「仲間がいる → 一人じゃない → 入ってみよう」
**対象**: LINE公式・Discord・Slack・Facebookグループ・オープンチャット等

**例文**:
```
同じことで悩んでる人が集まってる場所、作ってます。
プロフのLINEから来てください。

ここで話し合いたい人向けのグループあります。
希望する人はDMで。

一人でやるより仲間がいた方が続く。
入り口はプロフのリンクから。
```

**使い所**: 共感型・悩み系投稿の末尾。「仲間がいる」という安心感で参加意欲を引き出す。

**鉄則**: グループの「何人いるか」より「どんな人がいるか・何を話しているか」を示す。

---

### ④ イベント・セミナー・体験型

**心理レバー**: 「来てみようかな（重くない・試せる）」
**対象**: セミナー集客・ワークショップ・勉強会・無料体験・説明会

**例文**:
```
来月、これをテーマに話す機会があります。
来れる人はプロフのリンクから。

もし対面で聞きたい人は、○○に来てください。
詳細はプロフのリンクに。

1回だけ試してみたい人向けに体験の場を作ってます。
（リンク）
```

**使い所**: 集客したい時の投稿末尾。「来てください」ではなく「来れる人は」「来たい人は」という表現で任意感を出す。

**鉄則**: 「来てください」より「来れる人は」のほうが行動を引き出す（プレッシャーを外す）。

---

### ⑤ 専門家権威・次投稿期待型

**心理レバー**: 「次も読みたい → 今日中にフォロー」
**対象**: 連続投稿設計・テーマシリーズ展開・権威構築

**例文**:
```
次の投稿では、この続き（応用編）を書きます。
```

```
これに関連して、よく質問されることを来週書きます。
```

```
3年で変わったことを今週ずっと書いてます。
明日は○○について。
```

**使い所**: 連続投稿の各回末尾。シリーズ化することでフォロー動機を作る。

**鉄則**: 「何を書くか」をぼかしすぎない。「○○について」と対象を示すだけで読者の期待値が上がる。

---

### ⑥ 脚注スタイル（さらっと最小限）

**心理レバー**: 「説明しないことが一番自然」
**対象**: 全ジャンル可（特にバズ投稿の末尾に最適）

**例文**:
```
（投稿本文）
↓
▶︎ [リンク]
```

または

```
（投稿本文）

気になる人はこちら ▶︎ [リンク]
```

または

```
（投稿本文）

詳しくはこちら →（プロフのリンクから）
```

**使い所**: 教育型バズ投稿の末尾にさらっと置くだけ。文字数最小・押し付けゼロ。サービス案内・LP・LINE登録等への導線として汎用的に使える。

## 🎯 アカウント目的 × 最適CTAマトリクス

| アカウントの目的 | ターゲット心理 | 第一推奨 | 第二推奨 |
|---|---|---|---|
| サービス業の集客 | 相談するのは重い | ①問い合わせ誘導 | ⑥脚注 |
| ブランド・権威構築 | 詳しく知りたい | ②コンテンツ継続 | ⑤次投稿期待 |
| コミュニティ形成 | 仲間がほしい | ③グループ参加 | ②コンテンツ継続 |
| イベント集客 | 行ってみようかな | ④体験・セミナー | ⑥脚注 |
| 専門家・思想リーダー | またここに来たい | ⑤次投稿期待 | ②コンテンツ継続 |
| バズ末尾の補助 | 興味あれば | ⑥脚注 | ①問い合わせ誘導 |

## 💡 CTAの組み合わせルール

- **1投稿 = 1〜2CTAまで**（複数置くと押し付け感が出る）
- **同じパターンを週3回以上連続使用しない**（飽きられる）
- **バズ投稿（教育型）の末尾は ⑥脚注 が基本**
- **1万インプ超えた投稿には ①または③ を試してみる**（転換率が高い）

---

## 🔄 CTAの一貫性チェック

以下が揃っているか確認：

- [ ] プロフィールに「相談・問い合わせ先」「コミュニティ入口」が記載されている
- [ ] CTA で案内する先（LINE/リンク/DM）が実際に機能している
- [ ] 投稿ジャンルとCTAの目的が一致している（育児発信 × 転職CTAはズレ）
- [ ] 週1回以上、何らかのCTAが入っている

## 📊 効果測定の考え方

教育型投稿（プレイブック準拠）+ 目的別CTA の組み合わせで期待できる動線：

```
バズ教育投稿（インプ1万〜）
    ↓
プロフ流入（インプの0.5〜2%）
    ↓
CTA経由の接触（LINE登録・DM・問い合わせ）
```

インプが増えるほどCTAへの流入も増える。
**まず教育型プレイブックでバズを作り、そこにCTAを置く**という順序が正しい。
