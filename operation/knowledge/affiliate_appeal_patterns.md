# アフィリ訴求パターン6種類

「焦らせない・押し付けない・自然」を貫く訴求テンプレート集。
商品ジャンル × ターゲット心理に応じて使い分ける。

---

## 🚫 絶対NG表現（焦らせ・押し付け）

| NG表現 | 理由 |
|---|---|
| 「○日までセール」「明後日まで」「3日後」 | 後で見返した時に古い情報・煽り感 |
| 「今すぐ買って！」「急いで！」 | 押し付けがましい |
| 「在庫切れる前に」「数量限定」 | 焦らせ感 |
| 「気になる人はチェック」 | 定型句感が強い・古い |
| 「DMで聞かれたので置いとくね」 | まだ広告色強い・テンプレ感 |
| 「楽天セール前にチェック！」 | 期間訴求はOFF |
| 「絶対買って！」「リピート確定」 | 強引 |

---

## ✅ 採用する訴求パターン6種

### ① 体験告白型（うちで使ってる感）

**心理レバー**: 「自分の話だから押し付けじゃない」
**適合商品**: 絵本・おもちゃ・収納・キャラ系・育児雑貨

**例文**:
```
うちで使ってるのはこれ。
私はずっとこれ。
結局ここに戻ってきた。
我が家の定番。
何度買い替えてもこれに戻る。
```

**使い所**: フォロワーが「あ、この人使ってるんだ」と認識する程度の軽い置き方。

---

### ② 失敗→正解型（試行錯誤の結論）

**心理レバー**: 「失敗まで教えてくれる人への信頼」
**適合商品**: 食材宅配・時短家電・ハッカ油等の使用品・通信教育

**例文**:
```
3つ試して結局これに戻った。
最初は○○買って失敗した。今はこれ。
いろいろ買い直した結果、これが一番。
○○と△△を試した後、これが定番化した。
最初の半年は他社使ってたけど、今はこれ。
```

**使い所**: 「試行錯誤の説得力」を出す。レビューサイト感ゼロ。

---

### ③ 権威継承型（他者からのお墨付き）

**心理レバー**: 「自分推しじゃないから自然」
**適合商品**: 絵本・知育系・通信教育・学習系・保育用品

**例文**:
```
保育園で先輩に教えてもらって、それから5年使ってる。
同じ業界の仲間が皆使ってるやつ。
先生が園で使ってたから真似した。
ベテラン[同職]に「これだけは買え」と言われたやつ。
研修で講師がオススメしてた。
```

**使い所**: ペルソナの専門権威+第三者推薦のダブルレイヤー。最も売れる構造。

---

### ④ 横展開・シリーズ型（深掘りオファー）

**心理レバー**: 「もっと知りたい欲を満たす」
**適合商品**: 絵本シリーズ・知育玩具・キャラ系・コレクション商品

**例文**:
```
シリーズ全部泣けるから一緒に読んでほしい。
同じメーカーの他も気になってる。
これと組み合わせると最強。
このシリーズだけで本棚1段埋まる。
○○もあわせて読むとさらに深い。
```

**使い所**: 1商品紹介から複数クリックへ。クロスセル自然発生型。

---

### ⑤ 結果先出し型（変化のビフォアフ）

**心理レバー**: 「自分にも起きるかも」
**適合商品**: 寝具・冷感系・夜泣き対策・しつけ系・健康系

**例文**:
```
これ使い始めてから子（家族）の○○が変わった。
夜泣きが3日で減ったのはこれ。
朝の戦争が終わった理由はこれ。
寝かしつけが20分→5分になった。
保育園での集中力が上がった日から振り返ったらこれだった。
```

**使い所**: 強い動機。但し効能断定にならないよう「変わった」「気がする」「振り返ったら」など曖昧表現で景表法・薬機法回避。

---

### ⑥ 脚注スタイル（さらっと最小限）

**心理レバー**: 「説明しないことが一番自然」
**適合商品**: 全ジャンル可（特にバズ投稿の末尾に最適）

**例文**:
```
（投稿本文）
↓
▶︎ [リンク]
PR
```

または

```
（投稿本文）

ちなみにこれ▶︎ [リンク] PR
```

**使い所**: バズった教育投稿の末尾に置くだけ。文字数最小・押し付けゼロ。

## 🎯 商品ジャンル × 最適訴求マトリクス

| 商品ジャンル | ターゲット心理 | 第一推奨 | 第二推奨 |
|---|---|---|---|
| 絵本・知育系 | 子に良いものを与えたい | ③権威継承 | ④シリーズ |
| 暑さ・季節グッズ | 今困ってる | ⑤結果先出し | ②失敗→正解 |
| 食材宅配・時短系 | 自分も楽したい | ②失敗→正解 | ⑤結果先出し |
| 通信教育・学習系 | 子の将来不安 | ③権威継承 | ⑤結果先出し |
| 冷感寝具・睡眠系 | 夜泣き・不眠解決 | ⑤結果先出し | ①体験告白 |
| 収納・育児雑貨 | 家がスッキリしたい | ①体験告白 | ②失敗→正解 |
| おもちゃ・キャラ系 | 子が喜ぶものを買いたい | ①体験告白 | ④シリーズ |
| ハッカ油・薬用品 | 使い方知りたい | ②失敗→正解 | ⑤結果先出し |
| バズ投稿末尾の補助 | 興味あれば見てね | ⑥脚注 | - |

## 💡 訴求の組み合わせルール

- **1投稿 = 1〜2訴求まで**（3つ以上は押し付け感が出る）
- **同じ訴求を週2回以上連続使用しない**（飽きられる）
- **バズ投稿（教育型）の末尾は ⑥脚注 が基本**
- **アフィリ専用日（火・金・土 SLOT_2）は ①②③④⑤ から組み合わせ**

---

## 🔄 PR表記の配置（再確認）

- **メイン投稿** = 商品名+推奨を含む場合のみ PR 必要
- **中間リプ** = 商品名出さない教育コンテンツなら PR 不要
- **リンク投稿** = PR 必須・単独行 `PR`（# なし）

連続スレッドでもリンク投稿だけにPRがあれば実務上OK（厳密にはメインにも欲しいが、teaser化していれば不要）。
