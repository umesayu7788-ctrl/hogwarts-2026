# 🏆 高インプ投稿の「考え方とルール」（汎用プレイブック）

このファイルは特定のジャンル・ペルソナに依存しない、**普遍的な「投稿が伸びる原則」と「ルール」**を体系化したものです。
他プロジェクト（エージェントホグワーツ等）や第三者配布フォルダにそのまま使えます。

---

## ⚠️ このファイルの使い方（前提）

**「テンプレを丸コピして使う」のではなく、「考え方を理解して自分の言葉に翻訳する」ためのものです。**

- 各原則の **「なぜ効くか」** を理解する
- それを **自分のペルソナ・業種** に翻訳して適用する
- テンプレの形だけ真似ると、不自然なAI感が出てバズらない

→ ペルソナ確定がまだなら、`_persona_setup_guide.md`（同フォルダ）を先に進めてください。

---

| 投稿①（暴露型・テンプレ未準拠） | × | 30 | 0 |

# 🎓 4つの核となる考え方

## ① 「読者の頭の中の言葉」から始める

人は**自分が今思っている言葉**を見ると、立ち止まる。
- 「ニキビが治らない」と悩んでる人は「『ニキビが治らない』」で始まる投稿で止まる
- 「単価が上がらない」と思ってる人は「『単価が上がらない』」で止まる

**原則**: 投稿の最初の3秒は「これは自分のことだ」と思わせる勝負。

## ② 「数字」が信頼を瞬間的に作る

「たくさんの」「いっぱい」より、**具体的な数字**の方が10倍信頼される。
- 「○年、○○人を見てきて」
- 「○○社、○○人を担当して」

**原則**: 経験を数字に翻訳する。曖昧な表現は信頼を失う。

## ③ 「1つだけ」が記憶に残る

「3つあります」「5つのコツ」より、**「1つだけ」**の方が深く刺さる。
- 「これに効く返しが1つだけあった」
- 「共通する原因が1つだけあった」

**原則**: 焦点を絞る。複数列挙は読者の脳に残らない。

## ④ 「逆ケース」が保存・シェアを生む

「これをやると良い」だけでなく、「**逆にこれをやってる人は失敗する**」を入れる。
- 「逆に○○な家庭は」
- 「逆に△△と言う人は」

**原則**: 警鐘で「自分は当てはまる？」と気にさせる。保存率が一気に上がる。

# 📐 必勝8ステップ構造（実装ガイド）

この構造は **「型」ではなく「考え方の順序」** です。順序通り考えれば、自然と伸びる投稿になります。

## 全体フォーマット（3連スレッド）

```
【1/3 メイン投稿】
Step 1. 引用フック『○○』             ← 読者の心の声を最初に出す
Step 2. 数字権威「○年、○○人見てきて」  ← 信頼を瞬間的に作る
Step 3. 結果先出し「○○が1つだけあった」← 続きを読みたい欲

【2/3 リプ1】
Step 4. 答えは1つだけ『○○』           ← 焦点を絞る
Step 5. 理由＋なぜ効くか（短く）        ← 納得感を作る
Step 6. 逆ケース対比「逆に○○な人は」    ← 警鐘で記憶に残す

【3/3 リプ2】
Step 7. 自分の実践（家・現場・自分の体験）← 親近感・本物感
Step 8. 低コストCTA「今日1回だけ試して✨」← 行動誘導
```

---

## Step 1: 引用フック『○○』

**考え方**: 抽象的な見出しではなく、読者の頭の中にある**具体的な言葉**で始める。

**質問の立て方**: 「自分のフォロワーは今、頭の中でどんな言葉を呟いてる？」

**例（業種を変えて）**:
- 育児系: 『ママきらい』『うちの子起きない』
- 営業系: 『単価が上がらない』『リード来ない』
- 美容系: 『ニキビ治らない』『毛穴目立つ』
- 学習系: 『勉強しても上がらない』『集中力ない』
- 健康系: 『食べてないのに痩せない』『眠れない』

**NG**: 「○○のコツ」「○○な人の特徴」← 抽象的・自分ごとに感じない

---

## Step 2: 数字権威

**考え方**: 自分の経験を「数字」に翻訳する。

**質問の立て方**: 「自分は何年、何人/何社/何件と接してきた？」

**3パターン**:

| 種類 | 用途 | 例 |
|---|---|---|
| 個別経験（深い） | 1対1の関係・成長過程の話 | 「[専門家]歴○年、**○○○人を担当してきて**」 |
| 広域観察（広い） | 傾向・統計的観察 | 「○年、**○○人見てきて**」 |
| 現場頻度（日常） | 毎日の鉄板技 | 「**毎日○○人にやってる**現場で」 |

**自分の数字を出す方法**:
- 何年やってる？
- 1日/1週間/1ヶ月で何人と接する？
- それ × 年数 = 累計経験人数

**NG**: 「たくさんの」「いっぱい」「多くの」「長年」← 数字なしは弱い

---

## Step 3: 結果先出し

**考え方**: 「**1つだけ**あった」が最強フレーズ。

**ポイント**: 数を絞る方が強く刺さる。

**パターン**:
- 「○○に効く答えが**1つだけ**あった」
- 「共通する原因が**1つだけ**あった」
- 「最大の見落としが**1つ**あった」

**NG**: 「3つあります」「5つのコツ」← 焦点ぼやける

---

## Step 4: 答えは1つだけ

**考え方**: 列挙しない。1つに絞る。

**OK例**:
```
それは、『○○』。
```

**NG例**:
```
①○○
②△△
③□□
```
（教育情報としてはアリだが、必勝テンプレでは焦点散る）

---

## Step 5: 理由＋なぜ効くか

**考え方**: 「ふーん」で終わらせず「**なるほど**」を作る。

**構造**: 「○○すると、△△なる。だから□□になる」

**専門用語OK・むしろ推奨**:
- 育児系：脳科学・睡眠科学・愛着理論
- 営業系：行動経済学・交渉心理学
- 健康系：栄養学・生理学
- 学習系：認知心理学・記憶のメカニズム

**長さ**: 3-5行で要点だけ

---

## Step 6: 逆ケース対比

**考え方**: 「これをやると良い」だけでなく、**「これをやると失敗する」**もセットで言う。

**効果**: 保存・シェアが一気に増える。

**例**:
> 逆に「早く答えて！」「そうじゃないでしょ！」が口ぐせの家庭の子は、自分の感覚を信じられなくなる。

「ドキッ」とする読者が増える = 自分ごと化 = 保存される。

---

## Step 7: 自分の実践（最重要・差別化ポイント）

**考え方**: 「他人の話」じゃなく「**自分はこうした**」を入れる。

**これがないと**: 教科書みたいな投稿になり、AI感が出る。

**パターン別の例**:

### 個人発信者の場合（自分自身の体験）
- 「自分自身、これに気づいてから5kg減った」
- 「自分が応募者だった時もこれで通過率が変わった」
- 「私のニキビもこれで治った」
- 「独立直後、これだけで月収が3倍になった」

### 子育て発信者の場合（家での実践）
- 「家でも自分の子に試してから、宿題バトルが激減した」
- 「うちの娘にこれを言うようにしてから、寝つきが変わった」

### 専門家発信者の場合（クライアントの変化）
- 「自分の塾でこれをやってから、定期テスト平均が20点上がった」
- 「クライアント企業に提案して、3ヶ月で売上1.5倍」
- 「うちの店でやって、リピート率が2倍になった」

**鉄則**: **数値変化があれば必ず入れる**（「3日で」「1週間で」「3倍に」）

**NG**: 「みんなやってます」「世間ではよく言われる」← 自分の話じゃないとAI感

---

## Step 8: 低コストCTA

**考え方**: 行動コストが**最小**な提案にする。

**鉄則**: **「1回・1日・1分」**で行動コスト最小。

**OK**:
- 「今日1回だけ試してみて✨」
- 「明日1日だけやってみて✨」
- 「1分だけ意識してみて✨」

**NG**:
- 「ぜひ毎日続けてください」← 重い
- 「絶対やってください」← 強い
- 「フォローしてくれたら教えます」← 押し付け

# 🚫 ジャンル不問・絶対NGリスト

## 表現NG
| NG | 理由 |
|---|---|
| 「絶対」「必ず」「100%」 | 景表法・薬機法リスク |
| 「○月○日まで」「セール前に」 | 古情報感・煽り |
| 「今すぐ買って」「急いで」 | 押し付け |
| 「気になる人はチェック」 | 定型句感 |
| 「DMで聞かれたので置いとくね」 | 広告色強い |
| 「○○してください」「○○しましょう」 | 上から目線 |

## 口調NG
| NG | OK代替 |
|---|---|
| 「いいから片せ」 | 「いいから片付けて」 |
| 「○○ろや」 | 「○○したかった😇」 |
| 「知らんが？」 | 「知らないんだよね笑」 |

## 絵文字
| NG | OK |
|---|---|
| 🤭 | 🤣 / 💦 / ✨ |
| 🙏 | ✨ / 🥲 / 💭 |

## 文末
| NG | OK |
|---|---|
| 「〜の。」 | 「〜なんだよね」 |
| 「〜なの。」 | 「〜だったり」 |
| 「〜なんだ。」 | 「〜かもね」 |

## カッコ
- ❌ 地の文を「」で囲む（AI感）
- ✅ 会話の引用・固有名詞・1単語強調のみ

# 💡 ジャンル別・考え方の応用例

**注意：これらは「構造をコピーする」ためではなく、「同じ考え方が異なる業種でどう機能するか」を理解するためのサンプル。**

## A. ビジネス・営業系

```
【1/3】
『単価が上がらない』

営業歴12年、500社を担当してきて、
これに効く一言が1つだけあった。

【2/3】
それは、『○○』。
最初の打ち合わせでこれを言うだけで、相手の評価軸が変わる。
（理由：交渉心理学的に主導権が逆転するメカニズム）
逆に「安くやります」が口ぐせの人は、永遠に単価が上がらない。

【3/3】
自分も独立直後、これだけで月収が3倍になった。
明日の打ち合わせで1回試してみて✨
```

**この投稿の核**:
- 引用フック「『単価が上がらない』」← 営業マンの心の声
- 数字「営業歴12年、500社」
- 答え1つ「『○○』」
- 逆ケース「逆に『安くやります』が口ぐせの人は」
- 自分の実践「自分も独立直後、月収が3倍」← **必須**
- CTA「明日の打ち合わせで1回」← 低コスト

## B. ダイエット・健康系

```
【1/3】
『食べてないのに痩せない』

トレーナー10年、3000人を見てきて、
共通する原因が1つだけあった。

【2/3】
それは、『○○』。
（理由：基礎代謝・自律神経・睡眠のメカニズム）
逆に「カロリー制限」だけ頑張ってる人は、永遠に停滞する。

【3/3】
私自身、これに気づいてから5kg減った。
今夜1食だけ試してみて✨
```

## C. 学習・受験系

```
【1/3】
『勉強しても成績が上がらない』

塾講師15年、1200人を教えてきて、
共通する弱点が1つだけあった。

【2/3】
それは、『○○』。
（理由：記憶定着のメカニズム）
逆に「とにかく問題集を回す」子は、伸び悩む典型。

【3/3】
自分の塾でこれをやって、定期テスト平均20点上がった。
今日の勉強1時間だけ試してみて✨
```

## D. 美容・スキンケア系

```
【1/3】
『ニキビが治らない』

美容皮膚科10年、肌3000人を見てきて、
最大の見落としが1つだけあった。

【2/3】
それは、『○○』。
（理由：皮膚科学・常在菌バランス）
逆に「高い化粧水」だけ買ってる人は、永遠に治らない。

【3/3】
自分のニキビもこれで治った。
今夜の洗顔1回だけ試してみて✨
```

## E. キャリア・転職系

```
【1/3】
『面接で落ちる』

採用担当8年、5000人を面接してきて、
落ちる人の共通点が1つだけあった。

【2/3】
それは、『○○』。
（理由：印象形成の心理学）
逆に「経歴アピール」ばかりの人は、書類落ちで終わる。

【3/3】
自分が応募者だった時もこれで通過率が変わった。
次の面接1回だけ意識してみて✨
```

→ **業種・ジャンル変えても考え方は同じ**。「主語と専門知識」だけ自分のものに置き換えれば成立。

# 📦 アフィリ・収益化と組み合わせる場合

教育型必勝テンプレでフォロワーを積みつつ、別スロット（or 別日）でアフィリ商品紹介。

訴求パターン6種（`affiliate_appeal_patterns.md` 参照）：
- ①体験告白型「うちで使ってるのはこれ」
- ②失敗→正解型「3つ試して結局これに戻った」
- ③権威継承型「○○で先輩に教えてもらって」
- ④横展開・シリーズ型「シリーズ全部いい」
- ⑤結果先出し型「これ使い始めてから変わった」
- ⑥脚注スタイル「▶︎ [リンク] PR」

**アフィリ投稿の鉄則**: 焦らせ・期間訴求・押し付けは全部禁止。「これいいよ」と言わず「私はこれ使ってる」と言うだけ。

# 🔧 自分のプロジェクトへの移植手順

## 前提

このプレイブックを自分のプロジェクトで使う前に、以下が確定している必要があります：

- **ペルソナ**（中の人・キャラ名・看板コピー）

- **発信ジャンル**（育児／ビジネス／美容／健康／学習／キャリア等）
- **核となる発信軸**（何を伝えたいか・誰の悩みを解決するか）

→ ペルソナ未確定なら **`_persona_setup_guide.md`**（同フォルダ）を先に進めてください。

## Step 4: buzz_posts.md（ペルソナ・声定義）の確認

自分のプロジェクトの `buzz_posts.md` または相当ファイルに以下が固定情報として書かれているか確認：

## Step 5: テスト投稿（5本）

自動生成 or 手動生成で5本作成し、必勝テンプレ準拠度を確認：

- ✅ 引用フック『○○』が冒頭にあるか
- ✅ 数字権威があるか
- ✅ 「1つだけ」「1つあった」があるか
- ✅ 逆ケース対比があるか
- ✅ 自分の実践があるか
- ✅ 低コストCTAがあるか
- ✅ NG表現がないか

5本中4本以上クリアしてれば運用開始OK。

# 🎯 まとめ：このプレイブックの本質

**「型」ではなく「考え方」**。

1. **読者の頭の中の言葉から始める**（引用フック）
2. **数字で信頼を作る**（権威）
3. **答えは1つに絞る**（記憶）
4. **逆ケースで警鐘を鳴らす**（保存・シェア）
5. **自分の実践を必ず入れる**（差別化・本物感）
6. **行動コストは最小に**（CTA）

これに **NG表現の徹底排除**を組み合わせれば、ジャンル問わず再現可能。
